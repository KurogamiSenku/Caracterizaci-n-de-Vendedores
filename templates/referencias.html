Fuentes y Referencias del Proyecto

La solidez de cualquier investigación radica en sus fuentes. A continuación, se detallan los recursos bibliográficos y de datos que fundamentan este trabajo.

Fuentes Primarias y Bibliografía

Explora las publicaciones y conjuntos de datos clave que sirvieron como pilares para este estudio.

  • La librería Matplotlib | Aprende con Alf.

    Autor: Alberca, A. S.

    Publicación/Plataforma: Aprende Con Alf.

    Fecha: 4 de octubre de 2020.

    Acceder al Artículo

    Recurso didáctico para comprender el uso de Matplotlib, fundamental para la generación de gráficas en el proyecto.

  • La librería Pandas | Aprende con Alf.

    Autor: Alberca, A. S.

    Publicación/Plataforma: Aprende Con Alf.

    Fecha: 14 de junio de 2022.

    Acceder al Artículo

    Guía para el manejo y análisis de datos con la librería Pandas, esencial para la fase de ingeniería de datos del proyecto.

  • CARACTERIZACIÓN DE VENDEDORES INFORMALES DEL MUNICIPIO DE CHÍA - 2024

    Fuente: Datos Abiertos Colombia

    Fecha de Consulta: 8 de Noviembre de 2024 (o la fecha de tu última consulta)

    Acceder al Conjunto de Datos

    Este es el dataset principal utilizado para la caracterización y el entrenamiento del modelo predictivo. Contiene información sociodemográfica y económica de los vendedores informales en Chía, esencial para comprender sus perfiles de vulnerabilidad.

  • SMOTE: Synthetic Minority Over-sampling technique.

    Autores: Chawla, N. V., Bowyer, K. W., Hall, L. O., & Kegelmeyer, W. P.

    Publicación: Journal of Artificial Intelligence Research, 16, 321–357.

    Acceder al Artículo (DOI)

    Este es el trabajo original que introduce la técnica SMOTE (Synthetic Minority Over-sampling Technique), un pilar en el preprocesamiento de datos para balancear clases desequilibradas en Machine Learning, crucial para la fiabilidad de nuestro modelo.

  • NumPy: La biblioteca de Python más utilizada en Data Science.

    Fuente: DataScientest.

    Fecha: 30 de octubre de 2023.

    Acceder al Artículo

    Recurso sobre NumPy, una librería esencial para la computación numérica eficiente en Python, utilizada en la manipulación de datos del proyecto.

  • A Comprehensive Analysis of Synthetic Minority Oversampling Technique (SMOTE) for handling class imbalance.

    Autores: Elreedy, D., & Atiya, A. F.

    Publicación: Information Sciences, 505, 32–64.

    Acceder al Artículo (DOI)

    Este artículo proporciona un análisis exhaustivo de SMOTE, una técnica fundamental utilizada en este proyecto para el manejo del desequilibrio de clases en los datos, optimizando el rendimiento del modelo.

  • os — Interfaces misceláneas del sistema operativo.

    Fuente: Documentación de Python.

    Versión: 3.10.17 (o según la que uses).

    Acceder a la Documentación

    Referencia para el módulo `os` de Python, utilizado para interactuar con el sistema operativo, como la gestión de rutas de archivos.

  • pickle — Python object serialization.

    Fuente: Python Documentation.

    Versión: 3.13 (o según la que uses).

    Acceder a la Documentación

    Documentación de la librería `pickle`, utilizada para serializar y deserializar objetos Python, crucial para guardar y cargar el modelo entrenado.
